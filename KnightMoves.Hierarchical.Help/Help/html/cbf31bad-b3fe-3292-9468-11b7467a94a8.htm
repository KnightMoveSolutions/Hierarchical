For classes that implement this interface, this method passes each child of this node to the nodeProcessor provided recursively down the tree. It does not include this node.

Type: Boolean
True if every execution of the ProcessNode method returns true, false if at least one of the executions returns false.
Remarks
This method performs recursion for you. Provide a node processor of type ITreeNodeProcessor<T> and this method will recursively travel down the children of this node passing each child node it encounters down the tree into the ProcessNode method of the processor. The processor can then do its thing against each node. This method will NOT include this node. If you need to include this node as well then use ProcessTree(ITreeNodeProcessorT) instead.
