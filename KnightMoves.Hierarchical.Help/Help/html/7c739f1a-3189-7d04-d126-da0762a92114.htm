Remarks
Accepts a regular collection of ITreeNode<T> objects in the form of a List<T> collection and builds a hierarchical object model. The collection of objects are assumed to have a single root object, which the method identifies as the object with a null ParentId. If it does not find the root node by this criteria it will throw an ArgumentException. All other objects will be put in their respective place in the hierarchy as long as their ParentId values are specified.
