ITreeNodeT Methods

The ITreeNodeT generic type exposes the following members.

Methods
  Name Description
FindById
For classes that implement this interface, this method finds and returns the ITreeNode<T> object where the Id value is equal to the nodeId value provided as an argument. It will search the tree recursively until it is found.
IsAncestor
For classes that implement this interface, this method determines if the node provided as the treeNode is an ancestor of this node up the tree.
IsDescendent
For classes that implement this interface, this method determines if the node provided as the treeNode is a descendant of this node down the tree.
IsSibling
For classes that implement this interface, this method determines if the node provided as the treeNode is a sibling of this node.
MakeChildOfUpperSibling
For classes that implement this interface, this method makes this node a child of the sibling higher in the order of the Children collection of its parent.
MakeSiblingOfParent
For classes that implement this interface, this method makes this node a sibling of its parent.
MoveDownInSiblingOrder
For classes that implement this interface, this method moves this node down in the sibling order.
MoveUpInSiblingOrder
For classes that implement this interface, this method moves this node up in the sibling order.
ProcessChildren(FuncITreeNodeT, Boolean)
For classes that implement this interface, this method passes each child of this node to the nodeProcessor provided recursively down the tree. It does not include this node.
ProcessChildren(ITreeNodeProcessorT)
For classes that implement this interface, this method passes each child of this node to the nodeProcessor provided recursively down the tree. It does not include this node.
ProcessTree(FuncITreeNodeT, Boolean)
For classes that implement this interface, this method passes each child of this node to the nodeProcessor provided recursively down the tree. Unlike ProcessChildren, this method will start with (include) this node.
ProcessTree(ITreeNodeProcessorT)
For classes that implement this interface, this method passes each child of this node to the nodeProcessor provided recursively down the tree. Unlike ProcessChildren, this method will start with (include) this node.
