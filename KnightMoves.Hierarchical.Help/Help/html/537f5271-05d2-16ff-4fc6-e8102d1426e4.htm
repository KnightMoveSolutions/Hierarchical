public void MakeChildOfUpperSibling()

Implements

ITreeNodeTMakeChildOfUpperSibling
Remarks
This method is most useful when wiring up to a [RIGHT-ARROW] button that is common on many user interfaces that edit trees. For example, if you are viewing a tree and you see two items listed under the same parent, you click the second item to highlight it, then click the right-arrow, the item will look like it moved to the right and became a child of the sibling just above it in the tree. Executing this method provides that right-arrow functionality in the object model.
