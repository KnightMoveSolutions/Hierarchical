The delegate method that will process each node down the tree.

Return Value

Type: Boolean
True if every execution of the delegate method returns true, false if at least one of the executions returns false.
Remarks
This method performs recursion for you. Provide a delegate as a node processor of type that accepts an ITreeNode<T> object (which is for the nodes of the tree) and returns a bool as true if the execution of the delegate is successful or false if not. This method will recursively travel down the children of this node passing each child node it encounters down the tree into the delegate method. The delegate method can then do its thing against each node. Unlike ProcessChildren, this method will start with (include) this node. If you need to exclude this node then use ProcessChildren(FuncITreeNodeT, Boolean) instead.
