Remarks
This class is provided as a wrapper for entity classes that cannot directly inherit from TreeNode<> since there is no multiple inheritance in .NET (thankfully). Instead of inheriting from TreeNode<> the entity object can be wrapped by this class, which itself inherits from TreeNode<> on the entity object's behalf. Binding to values of the entity object requires accessing the object via the Entity property.
